### Inheritance in Dart
Inheritance means a class can use the properties and methods of another class.

## Key Idea
* Parent Class (Superclass) → gives properties & methods
* Child Class (Subclass) → receives and uses them

## How It Works
* Child class automatically gets:
  * Variables (properties)
  * Functions (methods)
* No need to rewrite code → saves time

```dart
class Father{
  String name = "Kabir";
  String property = "Flat";
  String incomesource = "Rent";
}

class Son extends Father{
  
}

void main(){
  Son azmir = Son();
  print(azmir.name);
  print(azmir.property);
  print(azmir.incomesource);
}
```

## Output

```sql
Kabir
Flat
Rent
```
