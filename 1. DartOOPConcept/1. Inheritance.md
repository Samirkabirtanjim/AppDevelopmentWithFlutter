### Inheritance in Dart
Inheritance means a class can use the properties and methods of another class.

## Key Idea
* Parent Class (Superclass) → gives properties & methods
* Child Class (Subclass) → receives and uses them

## How It Works
* Child class automatically gets:
  * Variables (properties)
  * Functions (methods)
* No need to rewrite code → saves time

```dart
class Father{
  String name = "Kabir";
  String property = "Flat";
  String incomesource = "Rent";
}

class Son extends Father{
  
}

void main(){
  Son azmir = Son();
  print(azmir.name);
  print(azmir.property);
  print(azmir.incomesource);
}
```

## Output

```sql
Kabir
Flat
Rent
```

### What’s happening here?
## Parent Class

```dart
class Father {
  String name = "Kabir";
  String property = "Flat";
  String incomesource = "Rent";
}
```

* Father class has:
 * name
 * property
 * income source
   
## Child Class

```dart
class Son extends Father {

}
```

* Son class inherits everything from Father using extends
* Even though Son is empty, it still gets all properties

## Main Function

```dart
void main() {
  Son azmir = Son();
  print(azmir.name);
  print(azmir.property);
  print(azmir.incomesource);
}
```

* You created an object azmir of Son
* But you are accessing:
 * name
 * property
 * incomesource
* These are not inside Son, they come from Father class

## Output
```sql
Kabir
Flat
Rent
```

## Key Understanding
* Son automatically got all data from Father
* No need to rewrite anything
* This is called pure inheritance
  
## Simple Real-Life Idea

* Son inherits from Father
So he gets:
* Name
* Property
* Income source
