# 📘 OOP in Dart — extends vs implements

---

# 🔷 Example 1: Using `extends`

```dart
abstract class Animal {
  String name;
  Animal(this.name);

  void eat() {
    print("${name} can Eat");
  }

  void makeSound() {
    print("${name} can make sound");
  }
}

class Cat extends Animal {
  String fly;

  Cat(super.name, this.fly);
}

void main() {
  Cat jerry = Cat("Jerry", "Can't fly");
  print("Cat Name: ${jerry.name}");
  print("Cat Fly: ${jerry.fly}");
  jerry.eat();
  jerry.makeSound();
}
```

## 🔷 1. What your code does

```dart
abstract class Animal {
  String name;
  Animal(this.name);

  void eat() {
    print("${name} can Eat");
  }

  void makeSound() {
    print("${name} can make sound");
  }
}
```

* Animal is abstract → cannot create objects directly
* It can have variables and methods
* Constructor initializes name
* Methods already have implementation

## 🔷 Child class

```dart
class Cat extends Animal {
  String fly;

  Cat(super.name, this.fly);
}
```

* Cat inherits from Animal
* Gets name, eat(), makeSound() automatically
* super.name → calls parent constructor
* Adds its own property fly

## 🔷 Main Function

```dart
Cat jerry = Cat("Jerry", "Can't fly");

print("Cat Name: ${jerry.name}");
print("Cat Fly: ${jerry.fly}");

jerry.eat();
jerry.makeSound();
```

## 🔷 Output

```sql
Cat Name: Jerry
Cat Fly: Can't fly
Jerry can Eat
Jerry can make sound
```

## 🔷 Important Notes

1. Abstract class

* ❌ Cannot do: Animal a = Animal("Dog");
* ✔ Used for inheritance
  
2. super.name
* Calls parent constructor
* Same as:
```dart
Cat(String name, this.fly) : super(name);
```

3. Inheritance
* Child reuses parent methods
4. Extra property
* fly exists only in Cat

## 🔷 Example 2: Using implements

```dart
class Person {
  void name() {}
  void age() {}
  void weight() {}
  void height() {}
}

class Student implements Person {
  @override
  void age() {
    print("Age is 12");
  }

  @override
  void height() {
    print("Height is 4.1 ft");
  }

  @override
  void name() {
    print("Name is MR.X");
  }

  @override
  void weight() {
    print("Weight is 30kg");
  }
}

void main() {
  Student x = Student();
  x.name();
  x.age();
  x.height();
  x.weight();
}
```
## 🔷 What implements means

```dart
class Student implements Person
```

* Person is used like an interface
* Student MUST implement all methods
